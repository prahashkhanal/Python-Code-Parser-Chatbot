📊How It Works

📦Analyze Repository

Clone and analyze a GitHub repository to extract code structure and dependencies.

🔍AI Project Summary

What it does: Uses AI to analyze the entire codebase and generate a human-readable summary.

💡 Includes: Project overview, key functions, data flow, and beginner-friendly explanations.

💬Ask AI Questions

What it does: Ask questions about the analyzed code and get AI-powered answers based on actual code context.

💡 Examples: "What does the main function do?", "Which functions are called the most?", "How does authentication work?"

📈Code Visualizations & Analysis

Explore your codebase through multiple visual perspectives to understand function relationships and dependencies.

📊 Call Graph

What it shows: A network diagram showing which functions call which other functions. Nodes represent functions, and arrows show call relationships.

💡 Use this to: Understand the overall architecture and how different parts of the code depend on each other.

🔄 Function Flow Chart

What it shows: A flowchart showing how functions call each other. Each box is a function, and arrows show the flow from one function to another.

💡 Use this to: Trace the execution path and see which functions are called in sequence.

📋 Click "Generate Flow Chart" to create a visual diagram

📋 Function List

What it shows: A complete table of all parsed functions and the other functions they call.

💡 Use this to: Search for specific functions and see their immediate dependencies in detail.
